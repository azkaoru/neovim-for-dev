# flash.nvim 利用ガイド

## 概要

`folke/flash.nvim` は Neovim 向けの高速ジャンプ・テキスト選択プラグインです。
Treesitter との連携により、構文ノード単位での選択・コピーが可能です。

> **注意**: `lightspeed.nvim` と `s`/`S` キーが競合するため、本設定では `lightspeed.nvim` を `enabled = false` で無効化しています。

---

## キーマッピング一覧

| キー | モード | 機能 |
|------|--------|------|
| `s` | n, x, o | 画面上の文字へジャンプ |
| `S` | n, x, o | Treesitter ノードを選択してジャンプ |
| `r` | o (operator-pending) | リモートフラッシュ（遠隔操作後に元位置へ戻る） |
| `R` | o, x | Treesitter ノードを検索して選択 |
| `<c-s>` | c (command) | Flash 検索の on/off 切り替え |

---

## 便利な利用手順

### 1. 通常ジャンプ（`s`）

画面上の任意の文字へ素早くジャンプします。

```
s{文字} → ラベルが表示される → ラベルキーを押してジャンプ
```

**例**: `sfu` → 画面上の `fu` にジャンプ

---

### 2. Treesitter ノード選択（`S`）

Treesitter で解析した構文ノード（関数・ブロック・引数など）を選択してジャンプします。

```
S → 構文ノードにラベルが表示される → ラベルキーでジャンプ
```

ビジュアルモードと組み合わせると、ノード全体を選択できます。

```
v → S → ラベル選択 → 構文ノード全体がビジュアル選択される
```

---

### 3. リモートヤンク + 元位置に戻る（`yr` + モーション）

離れた場所のテキストをコピーして、**カーソルを元の位置に戻します**。

```
y → r → {ラベル選択} → {モーション} → ヤンク → 元の位置に戻る → p
```

#### よく使うモーションの組み合わせ

| 操作 | キー |
|------|------|
| 1単語コピー | `yr{ラベル}w` |
| 行全体コピー | `yr{ラベル}yy` |
| 段落コピー | `yr{ラベル}ip` |
| 段落（空行含む）コピー | `yr{ラベル}ap` |

---

### 4. リモートヤンク + Treesitter ノードコピー（`yr` + `S`）

離れた場所の **Treesitter 構文ノード** をコピーして、カーソルを元の位置に戻します。

```
y → r → {ラベルでリモート位置へジャンプ} → S → {Treesitterラベル選択}
→ ヤンク → 元の位置に戻る → p
```

**使いどころ**: 関数定義・ブロック全体・引数リストなど、構文単位でコピーしたいとき。

---

### 5. Treesitter 検索ヤンク（`yR`）

Treesitter ノードを検索してヤンクします。
**カーソルはノード位置に移動します**（元位置には戻りません）。

```
y → R → {検索文字入力} → ラベル選択 → ヤンク → カーソルはそのノード位置に移動
```

**使いどころ**: ジャンプ先で作業を続けたいとき。

---

### 6. ビジュアルモードでの範囲拡張（`S`）

ビジュアル選択中に `S` で選択範囲を Treesitter ノード単位に拡張できます。

```
v → {任意の選択} → S → ラベル選択 → 選択範囲が構文ノード全体に拡張される
```

---

### 7. コマンドモードでの Flash 切り替え（`<c-s>`）

コマンドモード（`/` 検索など）で `<c-s>` を押すと Flash による検索ハイライトを切り替えられます。

```
/{検索ワード} → <c-s> → Flash ハイライト on/off
```

---

## 操作フローまとめ

```
【コピーして元位置に戻りたい】
  y → r → {ラベル} → ip（段落）/ w（単語）/ S（Treesitterノード）
  → p でペースト

【Treesitter ノードへジャンプしたい】
  S → {ラベル} → ジャンプ

【Treesitter ノードを検索してヤンク（位置移動あり）】
  y → R → {検索} → {ラベル} → ヤンク
```
